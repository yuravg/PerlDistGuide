# Example::Hello

A simple Perl package that prints "Hello World!" via an executable script.

## Installation
1. Install Dist::Zilla and dependencies (see guide).
2. Run `dzil build` to create the package.
3. Run `dzil install` to install it.

## Usage
Run the script:
```bash
example-hello
```
Output: `Hello World!`

## Testing
Run `dzil test` to verify functionality.
